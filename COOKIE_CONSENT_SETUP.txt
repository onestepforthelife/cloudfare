--------------
1. cookie-consent.js - Main consent logic

HOW TO ADD TO YOUR PAGES:
--------------------------

Add these lines to the <head> section of EVERY page:

<!-- Cookie Consent Banner -->
<link rel="stylesheet" href="cookie-consent.css">
<script src="cookie-consent.js"></script>

FEATURES:
---------

1. THREE CONSENT OPTIONS:
   - Accept All: Grants all permissions
   - Necessary Only: Denies optional cookies
   - Customize: User chooses specific categories

2. COOKIE CATEGORIES:
   - Analytics Cookies (Google Analytics)
   - Marketing Cookies (for future use)

3. GOOGLE CONSENT MODE V2:
   - analytics_storage
   - ad_storage
   - ad_user_data
   - ad_personalization

4. USER PREFERENCES:
   - Stored in localStorage
   - Persists across sessions
   - Can be reset anytime

TESTING:
--------

1. Open any page - banner should appear at bottom
2. Click "Accept All" - banner disappears, consent saved
3. Reload page - banner should NOT appear (consent remembered)
4. To test again, open browser console and type:
   resetCookieConsent()

GOOGLE ANALYTICS INTEGRATION:
------------------------------

The consent banner automatically:
1. Blocks Google Analytics until consent given
2. Updates consent mode when user chooses
3. Respects user preferences on every page load

Your Google Analytics will only track users who:
- Click "Accept All", OR
- Click "Customize" and check "Analytics Cookies"

CUSTOMIZATION:
--------------

To add more cookie categories, edit cookie-consent.js:

SUPPORT:
--------

If banner doesn't appear:
1. Check browser console for errors
2. Verify files are uploaded correctly
3. Ensure <script> tag is in <head> section
4. Clear browser cache and test

To reset consent for testing:
- Open browser console
- Type: resetCookieConsent()
- Press Enter
